# BRMedia ‘PC or Mobile’ Centre

This repository is the **central monorepo** for the BRMedia ecosystem.

It contains the full modular media workflow:

- 🎛 Converter — batch convert any media → any media  
- 🏷 Tagger — fast audio metadata editing  
- 🎚 Mastering — mastering + preview rendering  
- ▶️ Player — Evermusic-style streaming player  
- 🖥 Server — server-first streaming + APIs  

---

## 🧱 Architecture

This project uses a **monorepo structure**:

- `apps/` → End-user applications  
- `packages/` → Shared libraries  
- `server/` → Streaming + API backend  
- `tools/` → Dev utilities  

Everything is currently **blank scaffolding** ready for Phase 1 build.
